온라인 프로그래밍 캠프 개요

일정

  • 진행 기간 : 4월 6일 ~ 4월 24일(18일간)
  • 질문 업로드 시간 : 매주 화요일 오후 6시 ~9시까지
  • 준회원 질문 답변 : 매주 수요일 자정전 까지
  • 프로젝트 발표 : 매주 금요일
  • 일일 스크럼 : 매일(주말 제외)
  • 자율 개발 : 원하는 대로

당부 사항

  • 동료에게 물어보라
  • 동기를 도와주라
  • 실습은 개개인의 능동성과 자율
  • git 활용
  • PM(project manager) 지정 권장

Bankmanagement System (1주차 과제)

Fuction

  • menu()
    • welcome screen
  • new_acc()
    • 고객 계정 생성
    • 고객 정보 : 이름,생일,주민번호,주소,전화번호, 적금 기간?
  • view list()
    • 고객 계좌 정보 : 계정번호, 이름, 주소, 핸드폰 번호
  • edit()
    • 특정 고객의 정보 수정
  • transact()
    • 예금, 출금
  • erase()
    • 계정 삭제
  • see()
    • 고객 정보 : 계좌번호 이름 생일 주민번호 나이 주소 전화번호 계쩡 타입 저장된 돈 예금 날짜

break point

  • menu()

    • 계정 생성
    • 계정 정보 수정
    • 트랜젝션(예금,출금)
    • 계정 세부사항 확인
    • 계정 삭제
    • 계정 리스트 출력
    • 종료

  • new_acc()

    • checking accout, saving account 인가?

  • view list()

  • edit()

    • 어떤 정보를 수정할 것인지?

  • transact()

    • 어떤 계정인지? 분기보다는 break point?
    • 예금, 출금인가?

  • erase()

  • see()

개발 전 드는 생각

  • 생각보다 쉬울거 같다.
    • 객체 지향적으로 하면 쉬울 거 같단 생각이듬
    • 구조체 이용, 파일 입출력, 배열 3개면 충분
    • 참고할 코드도 많음

Python을 이용한 PC Speck check (3주차 과제)

목표

  • USB에 넣어서 해당 PC에 꽂은 후 실행시켜 엑셀에 데이터 저장 해보자[^1]

[^1]: 알바를 하는데 OS 버전이랑 메모리 용량을 확인 해야하는데 하루에 30~40대하니까 귀찮아가꼬 하핳

구현 사항

  • 프로그램 실행시 pc의 os, memory, hard disk usage가 엑셀에 저장된다
  • 각 행에 데이터들이 순차적으로 적용된다

미구현

  • 첫 번째 인덱스에 저장되어 있는 디스크의 정보만 받아옴
    • 동적으로 받아오진 못한다

아쉬움

  • API를 좀 더 찾아봤어야 하는데 거의 다 만들고 함수 찾아가꼬 헤맸다
    • 시간 낭비

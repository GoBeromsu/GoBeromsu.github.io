HTML

HTML TAG

맛보기

  • p : 시작 태그
  • id : 속성
  • id = “top” : 속성 값
  • class : 속성
  • class = “post” : 속성 값
  • 함 해보자구 : 내용

위에서 처럼 속성고 속성 값은 여러개 쓸 수 있다.

HTML TAG 종류

  • body
    • 웹 페이지에서 사용자에게 보여지는 본문 영역 시작과 끝을 정의하는 태그
    • html 한 단계 아래 영역에 존재
  • div
    • division : 분할, 구분
    • 본문 내 영역을 나누는 레이아웃 구성을 위해 주로 사용되는 태그
    • 크기 조절과 배경색 또는 배경이미지를 넣는 등의 효과를 줄 수 있다.
  • span
    • div 처럼 영역을 나누기는 하지만 주로 한 줄로 되어있는 텍스트를 구분 짓는다.
  • p
    • 주로 텍스트가 들어간다.
    • 단락을 표현하기 위한 태그
  • img
    • 사진을 표현하는 태그, 위치는 src 속성 값으로 설정한다.
    • 끝 태그가 없다.
  • ul, ol, li
    • ul : 순서 없는 목록
    • ol : 순서 있는 목록
    • 앞에 기호를 나타내어 목록을 표현을 위한 태그
    • 목록에 항목을 추가할 때마다 li 태그가 추가
  • input
    • 끝 태그가 없다.
    • 입력을 받는 요소를 표현하는 태그
  • a
    • anchor
    • 다른 웹페이지에 이동하기 위한 요소

CSS

css selecotr

  • p : selector
  • color : blue : 선언
    • color : 속성
    • blue : 속성 값

태그 선택자 (type selector)

  • 적용 기준 : 태그 값이 일치하는 요소

해당 웹 페이지의 모든 div, p 태그에 적용되는 것

클래스 선택자 (class selector)

  • 적용 기준 : class 값이 일치하는 요소
  • 선택자 정의 : .class 값

ID 선택자(ID selector)

  • 적용 기준 : id 값이 일치하는 요소
  • 선택자 정의 : #id 값

hexo Blog Rebuilding을 통해 점검 해봅시다
Created 2020-08-16| Updated 2020-08-17|BlogHexo
Post View:
  • 블로그 운영하면서 웃픈게 카테고리를 여러개로 만들다 보니 scaffold를 사용해도 직접 카테고리를 만들게 된다
    • 그러다보니 오타나면 갑자기 같은 카테고리가 대소문자 차이만 있고 2개 생기는 상황이 생긴다
    • 그런게 쌓이다가 정리안했더니 404 found 오류 뜨는 환경까지 내가 만드는 거지

hexo clean

  • hexo clean 명령어를 이용하면 public folder를 삭제한다.
    • 정적 사이트이다보니 md 파일을 html로 변환 후 그걸 뿌리는데 public folder에 저장되는거지

가끔 그런 상황에 사용하게 된다. 분명히 눈버깅해서 오류를 다 고쳤는데 블로그 게시물을 클릭 했을 때 404 not found 같은게 발생할 때 한 번 밀어주고 다시 배포하면 해결된다.

hexo list

  • hexo list [post|category|route|page]
    • 위의 명령어를 상룡하면된다
  • hexo route 같은 경우 블로그 스트럭쳐를 보여줘서 편하다
    • 블로그 점검할 때 사용 할 예정이다

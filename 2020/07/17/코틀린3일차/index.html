  • SimpleDateFormat()을 이용해 시간 포맷을 설정했더니 해결되었다.

listview listener 설정

  • ArrayAdapter로 리스트뷰와 배열을 연결한다

  • onItemClickListener를 정의한다

    • 클릭한 아이템의 포지션을 문자열로 받는다

  • mediaplay에서 prepare 하지 않으면 start 못함

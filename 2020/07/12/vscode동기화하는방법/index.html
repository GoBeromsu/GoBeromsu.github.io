vscode 동기화하는 방법

  1. Settings Sync 설치
  • Extension - Settings Sync 설치
  1. Github에서 token 받아온다
    • Settings - Developer settings - Personal access toekns 접근
    • 난 이미 vscode variables라고 만들어두었다.
    • Generate new token - gist 체크 -GENERATE TOKENS 클릭
    • 생성된 토크 COPY!
  2. VSCODE에서 토큰 등록
    • ctrl + p - settings.json - 아래 json 입력
  3. Shift + Alt + U 누르기
  • 세팅을 업로드하는 커맨드이다
  • 강제 실행할 것이냐 물을 건데 초기에는 강제 업로드해야한다. 걱정 놉
  • 세팅 다운로드는 shift + alt + d이다
  • 모든 결과는 출력 창에 출력된다

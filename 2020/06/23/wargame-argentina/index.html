blind-sql-injection 문제이다 한 번 풀어보자!

들어가면 포켓몬 볼 돌아가는 화면과 함께 키 입력 받는 폼이 있다. sql injection을 하려면 웹 페이지의 참 거짓을 봐야하므로 두 가지를 생각했다.

  1. 아무 키나 입력해서 false 화면 찾기
  2. key=1234or1=1을 입력해 참 화면을 찾기

위의 방식으로 참 거짓을 알아냈다. 이제 코드를 짜보자 블라인드 sql 인젝션 특성상 한 글자씩 넣어 알아내야한다.

위의 사진이 나오는 페이지 소스를 확인해보면 참 화면에 꺼내주셈이라는 텍스트가 있음을 알 수 있다. 그래서 꺼내주셈이라는 텍스트가 있는 화면이 나올 때만 코드를 출력하게 해서 키를 알아냈다.

플래그를 찾을 수 있다.

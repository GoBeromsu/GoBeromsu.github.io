jQuery 만들어진 이유

  1. 못생긴 코드
  2. 멍청한 브라우저!
  • 브라우저간 호환성이 좋지 않아 호환성 향상과 코드를 좀 더 깔꼼하게 짜기 위해서 만들어짐
  • 브라우저 호환성을 신경쓰지 않고 예쁜 코드를 짜보자

그런데 여기서 문제

  • js를 공부하지 않고 jQuery만 공부하는 현상이 생김

지금은 어떡해야할까?

  1. 못생긴 코드는 어떻게됬나?
    1. js가 섹시해졌다 업데이트와 함께!
  2. 브라우저는 어떤가?
    1. 호환성이 향상되었다

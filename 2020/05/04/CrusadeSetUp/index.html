공통 사항

Stage box

Stage box란?

  1. 믹서(console)와 Input(마이크, 악기) 거리가 멀어 중간 매개체로 이어주는 박스
  2. 센터의 경우 스테이지 박스에 XLR(cannon) 잭을 꽂으면 됩니다

Stage BOX 채널

  • A열과 C 열은 Input
  • I열은 Output
  • 상단 왼쪽 A열 부터 Input 1번~ 32번까지 채널이 32개이다
  • I열 하얀색 박스 왼쪽 부터 Output 1 ~ 8번이다

사용법

  • 건반 인풋은 12-13 채널이고 인이어는 3 번 채널입니다
    • 건반 output에서 나오는 선 2개를 12 -13번에 꽂고, 인이어와 연결된 선을 output 3번에 꽂아주세요
  • 간사님 마이크는 1번이고 나머지는 2번부터 5번까지 쓸게요!
    • 간사님 마이크는 input 1번에 나머지 싱어는 2 - 5 번까지 꽂으면 됩니다

싱어(Singer)

마이크

  • 마이크 소리가 작거나 안들린다고 마이크를 손으로 팡팡 치면 안되요!
    • 마이크가 고장날 수 있어요
    • 사람으로 치면 목소리가 작다고 머리를 내려치는 거에요
  • 마이크에 입을 너무 가까이 되면 안되요!
    • 입을 너무 가까이 데면 저음이 많아져 목소리가 불명확해져요
    • 채플 때 쓰는 마이크는 노래방 마이크랑 다른 마이크에요!
      • 입을 가까이 해도 노래방 마이크 느낌이 나지 않아요
  • 채플 전 후로 마이크에 선을 꽂거나 뽑을 때 뮤트 됬는지 확인해주세요!
    • 팡 소리가 날 수 있습니다
    • 엔지니어에게 뮤트 됬냐고 물어봐주세요!

위치

  • 스피커 앞에 서면 위험합니다
    • 스피커 소리가 마이크로 들어가 하울링이 생길 수 있어요

목소리 톤

  • 목소리 톤이 이상하다 느껴질 때 엔지니어에게 말해주세요
  • 요청하는 법
    • 아래로 내려갈수록 엔지니어가 문제를 해결하기 쉬워집니다!
    1. 목소리가 이상해요!
    2. 뭔가 ~ 느낌이에요
      • 너무 허스키해요
      • 시끄러운거 같아요
      • 찢어지는거 같아요
      • 목소리가 가벼운 느낌이에요
    3. ~ 영역이 ~한 거 같으니 ~ 해주세요
      • 목소리에 저음이 너무 많은 것 같아요
      • 하이가 너무 많은거 같아요

Crusade Setup
Created 2020-05-04| Updated 2020-05-09|Sound
Post View:

이 글 읽어 주셔서 감사합니다. 기본적으로 악기와 마이크 세팅하는 법, 평소 인이어 관련 궁금했을 것들을 적어 놓았습니다. 공통 사항과 맡은 파트 부분은 꼭 읽어주시면 감사합니다. 이 글을 읽는 다면 어디서든 본인 악기와 마이크를 세팅 할 수 있습니다.

공통 사항

XLR 잭이란?

  • 마이크 꽂을 때 자주 볼 수 있는 선입니다. 악기의 경우 D.I와 stagebox를 연결할 때 씁니다
  • XLR 잭의 뾰족 나온 부분을 숫놈, 들어가 있는 부분을 암놈이라 합니다
    • 숫놈은 stagebox의 INPUT에 꽂고, 암놈은 Output에 꽂습니다
  • 흔히 캐논 잭이라 이야기합니다

Stage box

Stage box란?

  1. 믹서(console)와 Input(마이크, 악기) 거리가 멀어 중간 매개체로 이어주는 박스
  2. 센터의 경우 스테이지 박스에 XLR(cannon) 잭을 꽂으면 됩니다

Stage BOX 채널

  • A열과 C 열은 Input
  • I열은 Output
  • 상단 왼쪽 A열 부터 Input 1번~ 32번까지 채널이 32개이다
  • I열 하얀색 박스 왼쪽 부터 Output 1 ~ 8번이다

사용법

  • 건반 인풋은 12-13 채널이고 인이어는 3 번 채널입니다
    • 건반 output에서 나오는 선 2개를 12 -13번에 꽂고, 인이어와 연결된 선을 output 3번에 꽂아주세요
  • 간사님 마이크는 1번이고 나머지는 2번부터 5번까지 쓸게요!
    • 간사님 마이크는 input 1번에 나머지 싱어는 2 - 5 번까지 꽂으면 됩니다

싱어(Singer)

마이크

  • 마이크 소리가 작거나 안들린다고 마이크를 손으로 팡팡 치면 안되요!
    • 마이크가 고장날 수 있어요
    • 사람으로 치면 목소리가 작다고 머리를 내려치는 거에요
  • 마이크에 입을 너무 가까이 되면 안되요!
    • 입을 너무 가까이 데면 저음이 많아져 목소리가 불명확해져요
    • 채플 때 쓰는 마이크는 노래방 마이크랑 다른 마이크에요!
      • 입을 가까이 해도 노래방 마이크 느낌이 나지 않아요
  • 채플 전 후로 마이크에 선을 꽂거나 뽑을 때 뮤트 됬는지 확인해주세요!
    • 팡 소리가 날 수 있습니다
    • 엔지니어에게 뮤트 됬냐고 물어봐주세요!

위치

  • 스피커 앞에 서면 위험합니다
    • 스피커 소리가 마이크로 들어가 하울링이 생길 수 있어요

목소리 톤

  • 목소리 톤이 이상하다 느껴질 때 엔지니어에게 말해주세요
  • 요청하는 법
    • 아래로 내려갈수록 엔지니어가 문제를 해결하기 쉬워집니다!
    1. 목소리가 이상해요!
    2. 뭔가 ~ 느낌이에요
      • 너무 허스키해요
      • 시끄러운거 같아요
      • 찢어지는거 같아요
      • 목소리가 가벼운 느낌이에요
    3. ~ 영역이 ~한 거 같으니 ~ 해주세요
      • 목소리에 저음이 너무 많은 것 같아요
      • 하이가 너무 많은거 같아요

악기(Session)

세션 공통 사항

인이어

  • 저희가 쓰는 인이어는 P2입니다

    • 아쉽게도 세션 입장에서는 볼륨 조절 밖에 하지 못합니다
    • 듣고 싶은 소리를 엔지니어에게 말해주세요!
    • 그래도 존재 자체가 감사한 친구죠!

  • P2 설치하는 법

    • 위에 말한 STAGEBOX Output에 꽂으면 됩니다

인이어 문제 발생 했을 때

위에 말씀드렸다시피 저희는 P2를 쓰니까 엔지가 인이어에 소스를 보내줘야 합니다 그래서 세션 입장에서 최소한으로 할 수 있는 것과 평소 왜 인이어가 안들리지? 같은 현상의 이유를 적겠습니다

소리가 안들릴 때 대응하는 법

  1. 인이어에 선이 연결되어 있는지 확인한다
  2. 볼륨이 키워져 있는지 확인한다
  3. 엔지니어에게 물어봅니다

소리가 찢어져요!

  • 이럴 때 찢어집니다
    • 인이어에 소리가 과하게 들어갈 경우
    • 소리 자체가 인이어가 감당할 수준을 벗어날 정도로 크게 출력되는 경우

소리가 이상해요 feat 답답하다, 톤이 이상하다

  • 인이어에는 이펙터(리버브)가 들어가지 않은 생 톤이 들어가 메인 스피커로 나오는 소리랑 다릅니다
  • 세션들이 모두 다른 이어폰을 쓰고 있습니다(엔지를 포함한)
    • 그렇기 때문에 이어폰마다 특징 때문에 같은 소리를 보내도 받는 사람은 다르게 듣습니다
  • 저희가 쓰는 P2 인이어의 한계이기도 합니다
    • P2 하나에 소리가 많이 몰리기 때문입니다
    • 그래서 듣고 싶은 소리를 다 받고 키우다보면 오히려 만족스러운 인이어 환경을 만들지 못합니다
      • 받는 소리(세션이나 싱어)(정확히는 주파수)가 겹칠 수 있고 겹치면 오히려 갑자기 잘들리던 내 악기 소리가 안들리는 거죠

소리가 갑자기 작아져요 Or 커져요

소리가 변하는 이유는 아래와 같습니다

  • 엔지가 나에게 보내는 소리의 크기(게인)을 변화시켰다
    • 보통 연습 시작 전 게인을 설정하고 왠만하면 만지지 않습니다
    • 만약에 예배 때나 연습 중에 만진다면 전체적인 밸런스 조정 중일겁니다
      • 엔지 입장에서도 초기 세팅 후에는 필요한 상황아니면 만지지 않습니다. 사고의 위험도 있고 게인을 만지면 티가 바로 나기 때문입니다
    • 소리가 너무 크거나 작으면 꼭 이야기해주세요 귀는 소중하니까요!
  • 소리를 메인 스피커로 듣고 있었다
    • 인이어를 한쪽만 빼고 듣거나 아니면 양쪽다 빼고 듣는 경우 일겁니다
    • 센터 특성 상 세션들 연습하는 장소에서 듣는 소리는 뒤에서 듣는 소리랑 다릅니다
      • 세션들 있는 장소에서 듣는 소리는 스피커에서 나온 소리가 뒤에 벽에 부딪히고 나오는 소리입니다. 부딪히면서 고음역대는 날라가고 저음이 돌아오는거죠 그래서 다른 소리를 듣는 겁니다
    • 중요
      • 귀 건강을 위해 인이어는 양 쪽 다 끼는게 좋습니다. 한쪽만 낄 경우 양 쪽 귀가 듣는 소리의 차이는 10db 정도입니다. 이 상태로 쭉 연습하고 예배 드리면 귀가 상합니다
        • 역시나 귀는 소중하니깐요
  • 심리적 요인
    • 집중도의 차이도 있고 기분 차이도 있습니다
    • 단 적인 예로 밥 먹고 났더니 갑자기 소리가 잘들리기도 하고 아까는 갑자기 들리던 소리가 안들리기도 합니다
    • 우리는 사람이기 때문에 어느 소리에 집중하느냐에 따라 같은 소리도 어떨 땐 크고 어떨 땐 작게 들립니다

악기 세팅

  • 기본적으로 드럼을 제외하고는 건반,신디,어쿠,일렉,베이스, 패드까지 모두 D.I를 씁니다
  • D.I의 Input에는 본인 악기의 Ouput을 꽂습니다.
  • D.I의 Output은 stagebox의 Input에 꽂습니다

정리하면 악기 - D.I - stagebox 이 흐름대로 연결되어야 합니다.

  • 소리가 저 순서대로 타고 믹서를 거쳐 출력 되는 것까지를 signal flow라 합니다

피아노(Piano)

  • 건반과 신디의 경우 stereo로 소리를 받습니다
    • 건반과 신디는 output이 2개, 건반 뒤에 보시면 ouput 적혀있고 L R 적혀 있는데 거기에 꽂으면 됩니다

신디

  • 신디의 경우 볼륨 페달 설치 할 때 헷갈릴 수 있습니다. 위의 사진이 볼륨페달입니다
    • 한 단계를 더 추가 해야합니다
    • 신디 - 페달 - D.I - Stagebox 순으로 연결이 되어야 합니다
    • 페달에 총 그래서 선이 4개 꽂혀 있어야 합니다.

베이스(Bass)

  • 일단 베이스는 mono입니다. 소리가 하나, 내 악기랑 연결된 선도 하나!
  • 베이스는 앰프랑 연결할 때 헷갈릴 수 있습니다. 저희 센터의 경우
    • 간단하게 설명하면, 베이스의 output은 D.I의 input으로, D.I의 Thru가 앰프 Input으로 가면 됩니다

드럼(Drum)

  • 드럼은 설치하고 마이킹이 되있어서 두는데 전자드럼이 난관입니다
  • 전자 드럼도 건반과 똑같이 stereo입니다, 선이 2개!
    • 전자 드럼 - D.I - StageBox 순으로 선을 꽂으면 됩니다
    • 드럼 뒤에 보면 output이라 적혀 있습니다

드럼 소형 믹서 설치하는 법

  • 왜 드럼 주자는 소형 믹서가 있는가?
    • 드럼 주자의 인이어
    • 메트로놈 소리를 받기 위해
  1. 핸드폰과 연결하는 선은 Aux 아니면 talkback out이라 적힌 곳에 꽂습니다
  2. 드럼 주자의 이어폰은 믹서 뒤에 Output이라 적힌 곳에 꽂습니다
    • 이어폰 꽂는 구멍이 있습니다
  3. 믹서 뒤 Output에 XLR 선을 꽂고 StageBox에 연결합니다
  • 엔지니어에게 물어보면 몇 번에 꽂아야 할지 가르쳐 줄거에요
  • 지금 꽂는 믹서의 output은 메트로놈 소리를 위해 꽂는 거에요

기타(acoustic)

  • 베이스랑 똑같습니다. 다른 점이 있다면 센터의 경우 앰프에 연결하지 않는다 정도?
    • 어쿠스틱 주자 입장에서는 기타 - D.I - Stagebox 순으로 연결하면 됩니다

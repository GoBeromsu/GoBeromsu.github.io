Python webcrawler를 만들어보자
Created 2020-05-06| Updated 2020-05-07|infosec119
Post View:

왜 갑자기 크롤러?

  • 파이썬으로 크롤링을 하려면 bs4라던가 selenium 라이브러리를 써야하는데 API 읽는 연습을 하고 싶었다
  • 크롤링하는거 1학년 때 하다가 못했는데 다시 해보고 싶기도 하고 재밌어보였다

Setup

  • 개발 환경 : VScode, python 3.8.2

  • 초기 세팅

  • python extension 설치

  • 가상환경 생성 및 활성화

  • 개발을 위한 모듈 설치
  • flake8 : python linting lib
  • black : python auto formatting lib
  • rope : python refactoring lib
  • ctrl + shift + P --> Python: Select Interpreter
    • 가상환경 골라줍니다

에러 사항

  1. vscode module not found error no module named 'selenium’과 같이 모듈을 찾지 못함
  • F1을 누른후 setting.json에서 python interpreter를 다시 잡아줌
  • 내가 설치한 python은 3.8인데 3.7.4가 돌아가고 있었음

실습

BeautifulSoup lib

API 분석?

기본적으로 bs4는 위의 코드 방식으로 움직인다. lib 안에 BeautifulSoup이라는 함수에 html 정보와 parser 종류를 인자로 받는다. 그 후 자기 맘대로 요리하는 거지

  • 일단 html 소스를 받아 크롤링하는 정적 툴
  • 태그를 읽어오는 방식이 2가지 있음
    • selecotr : css
    • find : html tag

예제 코드

  1. 날씨 정보 크롤링 해서 가져오는 것
  1. 내 블로그 게시물 긁어오는 크롤러

이거 땜에 삽질을 몇 시간 했는가. 왜 계속 syntax 오류 뜨지 했는데 리스트가 아니라 클래스로 정보를 긁어오고 거기서 접근한다. 사실 바로 눈치 챘어야 했다. 함수 실행되는걸 보고도 클래스 생각을 못하다니 아직 모자라다. 뭐 그거랑 openpyxl 여기서도 저번에 append 썼던 거 기억나는 대로 쓰다가 사용법 착각해서 헤맸다 뭐 그래도 나름 알아간게 있어서 좋다 뭔 헛짓이지

Kako Deveoper 시작해보자

  1. Kakao Developer 홈페이지에서 내 카카오 계정을 연동한다
  2. 내 어플리케이션에서 어플리케이션을 추가한다
    • 나는 이미 test용을 만들어서 있지만 어플리케이션 추가를 눌러 추가를 해야 카카오톡 API를 쓸 수 있다.
  3. 추가한 어플리케이션에 접속해 앱 키를 확인한다
  • 네이트 앱 키 : Android, ios, SDK에서 API를 호출할 때 사용한다
  • JavaScript 키 : Javascript SDK에서 API를 호출할 때 사용한다
  • REST API 키 : 역시나 REST API를 호출할 때 사용한다
  • Admin 키 : 모든 권한을 가지고 있는 키

사실 여기에 내 키를 올리는 것도 위험하긴 한데 test 용도이기도 하고 낌새 보이면 재발급 받으면 되니깐 공개하겠다

  1. 문서로 이동하여 API 보면서 개발 시작하면 끝!
  • API 깔꼼하게 잘되어 있다
  • API와 함께 가이드도 있고 가이드 문서에는 API 링크도 걸려 있어 비교적 이해하기 쉽다
  • 역시 갓카오인가?

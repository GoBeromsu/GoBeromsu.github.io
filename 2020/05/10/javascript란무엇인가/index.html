javascript란 무엇인가
Created 2020-05-10| Updated 2020-05-11|javascript
Post View:

Javascript

  • 초창기 javascript
    • 하나 이상의 이벤트를 웹에서 발생 시키기 위해서!
  • 현재에 이르러…
    • 웹이 빠르게 발전하면서 당연히 javascript의 가능성이 빠르게 커지고 있다

javascript version

  • ECMAscript == sepcification
    • ES5, ES6 등등
  • 중앙화가 되어 있어 하나가 업데이트되면 모든 브라우저에 업데이트 됨

Vanila javascript

  • 라이브러리가 없는 자바스크립트
    • 그래서 못생김
    • 빠르다!
    • 크기가 0 바이트 실화인가?
  • basic한 자바스크립트
    • 웹에서 기초가 되는 언어를 배우는 것이다
    • 웹에서 본 모든 것들이 바닐라 js로 이루어져 있음

Get It Start

  • css는 이런 느낌으로 적용하는거야
  • js파일을 바디 태그 아래에 넣어 주자
  • javascript의 helloworld
    • alert(), console.log()
  • 언어의 컨셉을 이해하자

문법

변수

  • 모든 intruction은 다른 줄에 선언되어야 한다
    • 그러고 싶지 않으면 ; 이용하자
  • 변수 생성, 초기화, 사용 단계로 사용하자
    • 기본이다!
  • const : 상수
  • let : 바뀌어도 되는 변수
  • variable : 변수
    • let처럼 바뀔 수 있음
  • 첫 사용은 const, 필요할 때만 let!

주석

  • //
    • 한 줄 주석
  • /* */
    • 여러 줄 주석

Data Type

Array

  • 데이터를 저장하는 곳
    • 리스트 느낌이지
  • indexing
    • daysOfWeek[2] --> sun

Object

  • object안에 object,array 모두 가능하다

Function

  • console.log()의 인자는 무한대로 받을 수 있다.

DOM(Document Object Module)

  • document로 javascript에서 css에 접근 할 수 있다.
  • JS에서 내 HTML의 모든 것들은 객체로 바꿔 버려 접근 가능하게 하는 것이지

Tips

`` 쓰는법

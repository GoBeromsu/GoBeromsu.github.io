문자

  • 문자는 ’ '나 " " 중 하나로 감싸야한다.

  • 문자를 더할 땐 + 사용

  • .length : 문자열의 길이를 계산하는 메소드

변수

  • variable의 줄임말로 변수 선언 할 때는 var [이름] 이렇게 사용한다
  • 유효 범위와 관계가 있다

연산자

  • = : 대입 연산자
  • == : 동등 연산자
    • 값이 같은가?
  • === : 일치 연산자
    • 좌우 항이 일치하는가?
  • != : 같지 않다
  • && : AND 연산자
  • || : OR 연산자
  • ! : 값을 역전시킴, 부정하는 것이지
    • 0 도 관습적으로 false가 될 수 있다

조건문

  • 사실 별 차이가 없다

반복문

  • document는 javascript를 이용해 웹페이지 문자를 적는데 콘솔에서 하면 안보인다
